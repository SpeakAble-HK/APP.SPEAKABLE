# SpeakAble 訂閱計劃定價

## 計劃概覽

### 家庭計劃 (Family Plan)
- **月費**: HK$129/月
- **年費**: HK$99/月 (慳 23%)
- **目標用戶**: 一般家庭，希望為孩子提供基礎粵語發音訓練

- **年費**: HK$199/月
- **目標用戶**: 需要深度分析、治療師協作、 unlimited 存取的高需求家庭
